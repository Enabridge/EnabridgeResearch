เรื่องที่ 1
ข่าวใหญ่ที่ทุก founder ใช้ MCP ต้องรู้คืนนี้. OX Security ปล่อย advisory ที่ตั้งชื่อแรงมาก The Mother of All AI Supply Chains. เนื้อหาคือ Model Context Protocol ของ Anthropic มี design flaw ที่ทำให้ arbitrary command execution บน server ที่ใช้ official SDK ทุกภาษา. Python, TypeScript, Java, Rust รวมกัน 150 ล้าน downloads. มี server public ออก internet 7,000 ตัวที่ scan เจอแล้ว. คาดทั้งหมด 200,000 instance.

OX โชว์ proof-of-concept ว่า execute command ได้บน 6 production platform จริง. รวม LiteLLM, LangChain, IBM LangFlow. ถ้า founder ไหน adopt MCP ใน production = เช็คทันที.

ที่เดือดกว่าคือ Anthropic ตอบกลับว่าไม่ใช่ bug. STDIO execution model เป็น secure default. การ sanitize input เป็นหน้าที่ของ developer ที่ deploy server เอง. จะไม่แก้ที่ protocol level.

ผลคือวงการแตกสองฝั่ง. ฝั่ง Anthropic ที่ปกป้อง design philosophy. ฝั่ง CISO กับ security researcher ที่บอกว่านี่คือ Log4Shell moment. เพราะ developer ส่วนใหญ่ assume ว่า SDK official ของ Anthropic = ปลอดภัย default. ความ asymmetry ตรงนี้คือ supply chain disaster.

ที่อยากให้ตามคือ trajectory ของ ecosystem. ถ้า Anthropic ไม่กลับลำใน 30 วัน. MCP จะเปิดช่องให้ Cloudflare, OpenAI Agents SDK, NVIDIA OpenShell ที่ market themselves ว่า secure-by-default fork ออกมา.

สำหรับ OpenBridge. Audit ทุก MCP server ที่ run หรือ depend on ภายในสัปดาห์นี้. และมี positioning opportunity ทันที ขายเป็น MCP gateway with security audit เหมือน Cloudflare ทำได้บน edge เราทำได้บน vertical stack. SMB ไทยจ่ายแน่นอน เพราะคำว่า MCP-native ตอนนี้แปลว่าเปิดช่อง RCE ในหู CISO.
