# Cognition AI คุยรอบใหม่ที่ valuation $25B — โต 2.5x จาก $10.2B ใน 7 เดือน, post-Windsurf $82M ARR + 350 enterprise: AI coding agent กลายเป็น category killer ที่ VC ยอมเทียบกับ Cursor/SpaceX deal

## TL;DR
- 23 เม.ย. Bloomberg/SiliconANGLE ยืนยัน Cognition AI (เจ้าของ Devin + Windsurf IDE) **คุยรอบ funding ใหม่ที่ valuation $25B** — โต 2.5x จาก $10.2B รอบ ก.ย. 2025 ใน 7 เดือน
- หลัง acquire Windsurf ก.ค. 2025 — รวมเป็น "agent + IDE" stack ที่มี **$82M ARR** (Windsurf side), **350+ enterprise customers**, ARR doubling QoQ
- Halo effect จาก SpaceX-Cursor $60B (เม.ย. 2026) ดึง investor demand — Founders Fund + Lux + Elad Gil น่าจะ double-down

## เกิดอะไรขึ้น

วันที่ 23 เม.ย. Bloomberg ปล่อย scoop ที่ทำให้ AI coding sector สั่นอีกครั้ง — **Cognition AI** บริษัทอายุ 24 เดือนที่สร้าง Devin (AI software engineer agent) **กำลังคุยรอบ funding ใหม่ที่ valuation $25 billion** ระดมเงิน "hundreds of millions" หรือมากกว่า. SiliconANGLE และ Techzine ตามด้วย confirmation ภายในชั่วโมงเดียวกัน — รอบนี้ยัง early talks, terms ยังเปลี่ยนได้, ไม่มี lead investor ประกาศ. แต่ตัวเลข $25B หมายถึง **2.5x markup จาก $10.2B** ที่ Cognition ปิดรอบ Founders Fund-led $400M ก.ย. 2025 — และ **6.25x markup จาก $4B** เม.ย. 2025 (12 เดือนก่อน).

ตัวเลขที่เปลี่ยน narrative คือ **ผลพวงของการ acquire Windsurf** ก.ค. 2025 (ดีลที่ Cognition คว้าหลังจาก Google จ่าย $2.4B license CEO Varun Mohan + ทีมหลักไป Google Brain, OpenAI ถอยหลังจากเสนอ $3B ซื้อ Windsurf ทั้งบริษัท). ส่วน "ที่เหลือ" ของ Windsurf ที่ Cognition ได้มา — **IDE + product + brand + 350+ enterprise customers + $82M ARR** (Windsurf alone, ARR doubling QoQ ตาม disclosure ของ Cognition ใน blog post). รวมกับ Devin agent ของ Cognition เดิม (ARR ไม่เปิด, แต่ Sacra ประเมินที่ ~$50-80M Q1 2026) = **stack ที่อาจรวมเกิน $150M ARR** เข้า Q2 2026 — และนี่คือเหตุผลที่ valuation $25B ดูสมเหตุสมผลแม้ markup จะหนัก (multiple ~150-160x ARR ซึ่งสูงแต่ไม่ทะลุเทียบ Cursor ที่ SpaceX จ่าย $60B ที่ ARR rumor ~$300-400M).

Halo effect ของ **SpaceX-Cursor $60B** (เม.ย. 2026 ที่ Yoh เคยอ่านเมื่อสัปดาห์ที่แล้ว) ดูเหมือนจะดึง investor demand เข้า Cognition โดยตรง — Bloomberg quote source ที่ใกล้ดีลบอกว่า "interest from investors has increased following the Cursor announcement" และ Founders Fund (lead รอบก่อน), Lux Capital, 8VC, Elad Gil, Definition Capital, Swish Ventures น่าจะ double down รอบนี้ + Sequoia/Tiger/Insight ที่ pass รอบก่อนน่าจะกลับมาคุย. ใน Twitter/X — **Scott Wu CEO Cognition** ทวีตหลัง Bloomberg ออกเพียง "the work continues. devin's been busy." ไม่ confirm/deny — pattern เดียวกับที่เขา handle Windsurf acquisition rumors ก.ค. 2025.

ตัวเลข strategic ที่ Cognition disclose ในปี 2026: **Devin ตอนนี้ทำ task SWE-bench Verified ได้ 76%** (เทียบ Claude 4.5 Sonnet 72%, GPT-5.5 78%, Cursor Composer 73%); **enterprise ARR ของ Windsurf** doubling Q1 → Q2 2026 (ทำให้ run-rate Q2 อาจแตะ $160M+ Windsurf-only); **350+ enterprise customers** ที่ deploy production (รวม Coinbase, Ramp, Vercel, Mercor, Nubank ตามที่ Cognition disclose); และ Devin agent ตัวเดียวเริ่ม handle **80%+ ของ pull request ที่ merge เข้า trunk** ใน 5 customer flagship — หมายถึง Devin ไม่ใช่ assistant แต่เป็น primary engineer ที่มนุษย์ review.

## ทำไมสำคัญ

นี่คือ **moment ที่ AI coding กลายเป็น category killer ที่ VC ยอม markup 2.5x ใน 7 เดือน** — สวนทางกับ thesis ปกติที่ AI infrastructure spending จะ commoditize ลงเรื่อยๆ. เปรียบเทียบ — Cursor ($60B), Cognition ($25B), GitHub Copilot (ถูก Microsoft กิน $7.5B ปี 2018 + ตอนนี้น่าจะ standalone $50B+), Replit ($3B), Codeium $1.25B (ก่อนถูก Cognition กิน) — **AI coding sector total valuation ตอนนี้ทะลุ $200B**. และที่สำคัญคือ — ARR multiples ยังไม่ลดลง ทั้งที่ มี 5+ player ใน frontier tier (Anthropic Claude Code, OpenAI Codex, Google Jules, Cursor, Cognition Devin, Replit Agent, Sourcegraph Cody Pro). ตลาดยอม pay premium เพราะ **switching cost ต่ำมากใน AI coding** — dev สามารถเปลี่ยน tool ในวันเดียว แต่ revenue capture ต่อ active dev สูงมาก ($30-100/เดือน × 500K-2M paying dev เป็น TAM ที่ใหญ่).

ที่ smart ของ Cognition คือ **dual-track strategy** — Devin (agent คุยกับ task ผ่าน chat, async, deploy เอง) + Windsurf IDE (sync IDE ให้ dev expert ทำ pair programming). Cursor มี IDE-only — Devin มี agent-only — Cognition มีทั้งสอง = **catch ทั้ง junior dev (ที่อยาก agent ทำให้) + senior dev (ที่อยาก IDE ที่เข้าใจ context ทั้ง repo)**. นี่คือ wedge ที่ unique จริง — ไม่มีใครใน frontier coding tier มี dual asset แบบนี้. และเพราะ Windsurf มี $82M ARR + 350 enterprise customer มาแล้วก่อน acquire — ไม่ใช่ pre-revenue bet.

ที่น่ากังวลคือ **runway การ markup 2.5x ใน 7 เดือนซ้ำได้แค่ไหน?** ถ้า Cognition ปิดรอบที่ $25B กลางปี 2026 — รอบถัดไปต้องอย่างน้อย $50B-75B ภายใน Q1 2027 (markup 2-3x) เพื่อ justify investor IRR. คำถามคือ ARR ต้องวิ่งจาก ~$160M Q2 → $500M+ Q1 2027 (3.1x in 9 months) — เป็น growth rate ที่ doable แต่ unforgiving ถ้า miss. ถ้า miss = down round = **AI coding sector cooling** = Cursor ที่ SpaceX ผูกไว้ที่ $60B จะดูแพง. ดังนั้น Q3-Q4 2026 ARR disclosure ของ Cognition + Cursor + GitHub Copilot จะเป็น **canary ของ tier นี้ทั้งหมด**.

## มุม OpenBridge

**Direct relevance ปานกลาง — แต่ implication strategic หนัก**. Cognition+Cursor+GitHub Copilot = **AI coding tier ที่ลูกค้า dev จ่ายโดยตรง ไม่ผ่าน enterprise procurement** — pattern ที่ OpenBridge ไม่ได้เล่น (OpenBridge เล่น integration tier ที่ขาย enterprise procurement). แต่ **2 implication ที่ OpenBridge ต้อง pay attention**:

(1) **Devin/Cursor/Copilot ตอนนี้สร้าง MCP server + A2A agent ได้เร็วมาก** — dev สามารถใช้ Devin generate "OpenBridge skill" สำหรับ FlowAccount/PEAK ภายใน 1-2 ชั่วโมง. หมายความว่า **moat ของ OpenBridge ไม่ใช่ "เราสร้าง integration ได้"** — moat ต้องเป็น **"เราเข้าใจ Thai SaaS context ลึกที่ AI coding ไม่ generate ออกมาเอง"** (PDPA edge case, Thai date format quirks ใน FlowAccount, K-Bank API throttling pattern, LINE OA broadcast quota rules ที่เปลี่ยนรายไตรมาส). OpenBridge ต้อง **ลงทุน document Thai SaaS quirks ใน knowledge base ที่ accessible เป็น MCP server** — ภายใน 60 วัน — กลายเป็น "Thai SaaS integration intelligence layer" ที่ Cognition Devin จะ call ก่อนที่จะ generate code.
