# Avoca แตะ unicorn ที่ $1B — vertical voice agent สำหรับช่างประปา/HVAC โบ๊ค $1B งานทั้งปี: agent SMB ทำ enterprise revenue ได้แล้ว

## TL;DR
- 27 เม.ย. — Avoca ระดมทุนรวม $125M+ ข้าม seed/A/B ที่ valuation $1B; Series B นำโดย Meritech + General Catalyst, Series A นำโดย Kleiner Perkins, ผู้ลงทุนเดิมรวม Amplify Partners + Y Combinator
- บริษัทขาย **AI voice agent** ที่รับ missed call 24/7 ให้ HVAC, plumbing, automotive, moving, roofing service และอื่น ๆ; Schedule, follow-up, custom marketing campaign — ทุกอย่าง autonomous; **on track booking $1B in jobs ปีนี้** (ไม่ใช่ revenue ของ Avoca แต่เป็น GMV ที่ลูกค้าได้จาก lead ที่ agent ปิด)
- **Pattern ที่ใหญ่กว่าตัวเลข:** ช่างประปาใน Phoenix กลายเป็น enterprise AI customer — vertical agent ที่ train บน workflow industry เดียว ขายให้ operator เล็กที่ไม่เคยคิดว่าตัวเองคือ AI buyer = ตลาด greenfield ที่ frontier lab + horizontal SaaS ทุกค่ายเข้าไม่ถึง

## เกิดอะไรขึ้น

วันที่ 27 เมษายน 2026 startup ชื่อ Avoca ที่ตั้งอยู่ใน San Francisco ประกาศว่า raise มาแล้วรวมกัน $125M+ ข้าม seed, Series A, และ Series B ที่ post-money valuation $1B กลายเป็น **vertical AI unicorn ใหม่ล่าสุด** ใน category ที่หลายคนยังไม่ทันสังเกต Series B รอบล่าสุดนำโดย Meritech Capital ร่วมกับ General Catalyst, Series A ก่อนหน้านำโดย Kleiner Perkins, ผู้ลงทุนเดิมรวม Amplify Partners และ Y Combinator (Avoca เป็น YC alum รุ่น Winter 2023)

Avoca ทำสิ่งเดียว — รับโทรศัพท์แทนช่างประปา ช่าง HVAC ช่างซ่อม โรงเรียนสอนขับรถ บริษัทขนของย้าย และ operator ของ "America's services economy" ที่อื่น ๆ โดย agent voice ของ Avoca ทำงาน 24/7 รับ missed call นัดเวลา ส่ง follow-up SMS รัน custom marketing campaign กับ lead ที่ยังไม่ปิด ทั้งหมดเป็น autonomous — เจ้าของกิจการที่เคยพลาด 30-40% ของ inbound call ในเวลาทำงาน + 80% ของ call ตอนกลางคืน ตอนนี้รับสายได้ 100% และ Avoca claim ว่าระบบสามารถปิดดีลในระดับที่ technician มืออาชีพปิดได้

ตัวเลข business เด่นที่สุดในประกาศคือ — Avoca อยู่ใน track ที่จะ **book $1B in jobs ปีนี้** ตัวเลขนี้ไม่ใช่ revenue ของ Avoca แต่เป็น GMV (gross merchandise value) ของ job ที่ลูกค้าได้จาก lead ที่ agent ของ Avoca รับเข้ามาและปิดได้ — Avoca เก็บ subscription monthly ตามปริมาณ call (ราคาเปิดเผยน้อย, สื่อ Fortune ระบุว่าค่าเฉลี่ยอยู่ราว ๆ $300–$1,500 ต่อเดือนต่อ branch ขึ้นกับ volume) แต่ value prop ที่ขายลูกค้าคือ "ช่างประปาในเมือง Phoenix ที่จ้างตัว Avoca รายเดือนไม่ถึงพันเหรียญ ได้ booking job เพิ่มในระดับ $30K–$100K ต่อเดือน"

CEO ของ Avoca เคยเล่าใน Fortune ว่า idea นี้เกิดจาก "chance encounter in Texas" ที่เพื่อนของ founder Tyler Chen เป็นช่างประปาบ่นว่าพลาด lead ทุกวันเพราะเอาคนรับโทรไม่ทัน — ทีม Avoca ตอนนั้น (Chen + Shrivastava + Braswell) เลยลองสร้าง voice agent prototype ใน YC Winter 2023 และเริ่มขายให้ HVAC + plumbing operator ใน Texas ก่อนขยาย ตอนระดมทุน Series B Avoca claim ว่ามี customer มากกว่า 4,000 service business ใน 50 states (ตัวเลขนี้สื่อยังไม่ verified ตรงจาก customer reference)

ที่น่าสังเกตคือ Avoca **ไม่ใช้ frontier model ของตัวเอง** — ใช้ off-the-shelf voice AI infrastructure (Vapi, Retell AI, ElevenLabs) ผสมกับ in-house orchestration + workflow knowledge สะสม value prop คือ "เรารู้ว่าช่างประปาคุยกับลูกค้าอย่างไร" — ไม่ใช่ "เรามีโมเดลใหญ่กว่า" — เป็น vertical knowledge moat แบบ pure-play

## ทำไมสำคัญ

ตลาด AI ใน 2026 ส่วนใหญ่จะอ่านข่าวนี้ผ่านไปเร็ว ๆ เพราะ "ช่างประปา" ดูเล็ก เทียบ Anthropic $380B ไม่ได้ — แต่ Avoca คือ **proof point แรก ๆ ของ thesis ที่ a16z + Bessemer เขียนไว้ตั้งแต่ปลาย 2024**: vertical AI agent สำหรับ SMB ที่ไม่เคยซื้อ enterprise software จะเป็นตลาด greenfield ที่ใหญ่กว่า horizontal AI tooling — และ Avoca เพิ่งจะ validate ว่ามี willingness to pay จริง + retention เพียงพอที่ Meritech (ที่เด่นด้าน enterprise SaaS growth round) ยอมเข้ามา lead Series B ที่ $1B

Pattern ที่เห็นชัด: **vertical agent กิน workflow ที่ horizontal tool ของยักษ์เข้าไม่ถึง** Salesforce ขายให้ HVAC operator $50/mo per user ไม่ได้เพราะ TAM ที่ Salesforce target คือ Fortune 5000 ที่ user > 100 คน Avoca ขายให้ shop ที่มี technician 3 คน + receptionist 1 คน ซึ่งรวมกัน 850,000 service operator ทั่วสหรัฐ — TAM ใหญ่มากแต่ revenue per customer ต่ำพอที่จะไม่ดึงดูด Salesforce/ServiceNow แต่สูงพอที่ Avoca focused เก็บได้ทำกำไรงาม

อีก signal ที่สำคัญ: Avoca ไม่ได้ build foundation model — ใช้ Vapi + Retell + ElevenLabs ที่เป็น infrastructure layer สาธารณะ + ชั้น orchestration + domain knowledge เป็น moat ของตัวเอง model นี้ใกล้เคียงกับ Cursor + Sierra ที่ว่าตัว frontier model ราคาตกลง 90%+ ใน 2 ปี ทำให้ AI startup ที่ "ชนะ workflow ของหนึ่งอุตสาหกรรม" สามารถสร้าง business model ที่กำไรงามได้โดยไม่ต้อง burn capital เป็นพันล้านบน compute เหมือน frontier lab; **moat คือ vertical knowledge + sales motion ไม่ใช่ model weights** — และ Avoca คือ replicable formula ที่อีก 50–100 vertical agent unicorn จะเดินตามใน 18 เดือนข้างหน้า

ความเสี่ยงที่ต้องจับตา: vertical voice agent มี barrier to entry ต่ำที่ infrastructure layer (Vapi + ElevenLabs ขายให้ใครก็ได้) — competitive moat ของ Avoca อยู่ที่ "เริ่มก่อน + sales motion เก่ง" และถ้า 11x หรือคู่แข่งใหม่ลงสนามด้วย discount aggressive 30-40% อาจกัด margin ได้เร็ว ในรอบ 12-18 เดือน — ตลาดนี้จึงต้องการ category leader ที่มี customer count ก่อนคู่แข่ง catch up

## มุม OpenBridge

ใกล้เคียงมาก แต่ยังไม่ direct: **ตลาด vertical voice agent ของไทย/ASEAN ยังเป็น greenfield สมบูรณ์** — ในไทยมี service operator (ช่างแอร์, ช่างประปา, ช่างไฟ, ขนของย้าย, automotive service center) เป็นแสน ๆ shop ที่พลาด lead ทุกวันเพราะรับโทรไม่ทัน, ใช้ LINE Official Account + IG DM ตอบช้า, ไม่มี CRM, ไม่ track conversion — ตลาดนี้ Avoca เข้ายากเพราะภาษา + culture + payment integration ที่ต่างกันสิ้นเชิง (ลูกค้าไทยจ่ายผ่าน PromptPay, ส่ง location ผ่าน LINE, ใช้รูปและเสียงไทยที่ Vapi/Retell ไม่ optimal)

OpenBridge มี option สอง path: (1) **build vertical agent ของตัวเอง** สำหรับ Thai service economy โดย partner กับ infrastructure (LINE official, ElevenLabs Thai voice, Looloo Voice) — risk คือ sales motion + GTM ต้องเรียนใหม่; (2) **เป็น orchestration layer ให้ vertical agent ของคนอื่น** ที่ build เฉพาะ industry — Tridge สำหรับเกษตร, Chic สำหรับ beauty service, ค่ายไหนก็ได้ที่ทำ vertical — โดย OpenBridge ขาย integration + payment + reporting layer ที่ทุก vertical agent ต้องการเหมือนกัน

ผมแนะนำ path 2 + selective path 1: pick 1-2 vertical (ลอง HVAC/aircon repair กับ moving service เพราะ TAM ไทยใหญ่) ทำ vertical agent ของตัวเองให้เห็น proof point — ขณะเดียวกันเปิด API + connector layer ให้ vertical builder รายเล็กที่ไม่อยาก build infrastructure เอง

อีกประเด็น: Avoca ระดมทุน $125M+ ที่ $1B โดยไม่มี foundation model ของตัวเอง — เป็น signal สำหรับ founder ASEAN ว่า "ไม่ต้อง build model" เป็น valid path ที่ VC ระดับ Meritech/General Catalyst/Kleiner Perkins ยอมเข้า — **Thai/ASEAN VC ควร rethink thesis ที่ขาย "เราจะ train Thai LLM"** ไปเป็น "เราจะชนะ vertical workflow X ใน Thailand/ASEAN" และ OpenBridge มี position ที่จะเป็น default infrastructure layer ของ vertical agent ASEAN ทุก vertical
