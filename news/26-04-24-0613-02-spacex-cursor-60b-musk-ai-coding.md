# SpaceX ดีลซื้อ Cursor $60B หรือจ่าย $10B ต่อ "ความร่วมมือ" — Musk ดักหน้า Microsoft, Cursor ยกเลิก round $2B/$50B valuation ทิ้ง 24 ชั่วโมง

## TL;DR
- **21–22 เม.ย.** SpaceX ประกาศมีสิทธิ์ซื้อ Cursor ที่ **$60B ภายในปลายปี** หรือจ่าย **$10B floor** สำหรับ "joint work" — preempt round $2B valuation $50B ที่ Cursor กำลังจะปิดกับ a16z/Nvidia/Thrive เหลือเวลาอีกไม่กี่ชั่วโมง
- **Microsoft ร่วมเจรจาซื้อ Cursor ก่อนหน้า** (Axios, CNBC) แต่ SpaceX เสนอ all-cash premium + **Colossus supercomputer 1M H100-equivalents** สำหรับ training — Cursor CEO Michael Truell เลือก Musk
- Cursor **revenue annualized $1B+** ใน 2 ปี — ลูกค้า millions of developers + thousands of enterprises; SpaceX deal condition = ปิดหลัง **IPO summer 2026** เพื่อใช้ stock ซื้อ
- Signal: **AI coding = strategic asset tier 1** สำหรับ frontier lab — Musk ยอมจ่าย 60x revenue multiple เพื่อคู่ compute + distribution ที่ XAI Grok ยังไม่มี; เริ่มต้น consolidation wave ของ AI coding

## เกิดอะไรขึ้น

เย็นวันอังคาร 21 เม.ย. ทีม Cursor ที่ออฟฟิศ SoHo กำลังอยู่ปลายทาง — Sequoia, a16z, Nvidia, Thrive Capital ตกลงแล้วที่ round $2B บน **$50B valuation** (3x จาก $16B ที่ rais เดือนต.ค. 2025). Michael Truell CEO เตรียมเซ็น term sheet เช้าวันพุธ. แล้วโทรเข้าจาก Gwynne Shotwell

จากรายงาน Bloomberg และ CNBC — SpaceX ยื่นข้อเสนอที่ไม่มีใครแข่งได้: **option ซื้อ Cursor ทั้งบริษัทที่ $60B ปลายปี 2026** หลัง SpaceX IPO summer (เพื่อใช้ publicly traded stock), **หรือจ่าย floor $10B cash** ถ้าไม่ exercise option ภายใต้ "joint work agreement". Musk ระบุเป้าหมายตรง: "combination of Cursor's leading product and distribution to expert software engineers with SpaceX's **million H100 equivalent Colossus training supercomputer** will allow them to build the world's most useful models". Truell เลือก Musk ภายใน 8 ชั่วโมง — round $2B ถูกทิ้งกลางทาง

Axios และ CNBC รายงานเพิ่มเย็นวันพฤหัสที่ **Microsoft เคยจ่อซื้อ Cursor มาก่อน** ตั้งแต่ต้นปี 2026 ผ่าน Satya Nadella + Kevin Scott ที่ push ว่า Cursor จะเติม Copilot story ที่ยังอ่อน. แต่ข้อเสนอ Microsoft ที่ตัวเลขไม่เปิดเผย (น่าจะ $40–50B equity + cash) ตีไม่ทันเพราะ regulatory timeline + GitHub Copilot antitrust overhang. Musk/SpaceX ไม่มีภาระนั้น เสนอ 60x revenue multiple ตรง ๆ

ตัวเลขของ Cursor ก็ช่วยอธิบาย pricing: **annualized revenue $1B+** (reported by Bloomberg), **millions of developers** ใช้ทุกวัน, **thousands of enterprises** เป็นลูกค้า รวม Fortune 100 banks, pharma, DoD contractors. Growth rate 300%+ YoY ภายใต้ GPT-5.5/Claude Opus 4.7 as backend. ถ้าดูเฉพาะ forward revenue 2027 ($3–5B projection) — 60x ยังดูสมเหตุสมผลใน AI frontier tier

ที่ซับซ้อนคือ deal structure. SpaceX **ชะลอ acquisition จนกว่าจะ IPO summer** เหตุผล CFO Bret Johnsen: "we want to avoid updating confidential S-1 filings before listing, and financing $60B via publicly traded stock is cleaner than cash drain on Starship/Starlink CapEx". ระหว่างนี้ Cursor ยังอิสระ operate — แต่ access Colossus compute เริ่มทันที, Musk นั่ง observer on board. TechCrunch เรียกดีลนี้ว่า **"gutsy Hail Mary"** — Musk ซื้อเวลาในเกม AI coding ที่ XAI/Grok ยังห่างไกล

## ทำไมสำคัญ

Pattern ที่ต้องอ่านคือ **AI coding = strategic tier 1 asset สำหรับ frontier lab**. เหตุผลไม่ใช่แค่ revenue — Cursor มี **data flywheel** ที่ยากจะ replicate: session logs ของ developer millions ของคน ที่ edit code จริง, accept/reject agent suggestion, debug ด้วย agent — data set นี้คือ training signal สำหรับ next-gen agentic model ที่ไม่สามารถซื้อจาก web scraping ได้. Musk ที่ขาด enterprise developer distribution (XAI มี consumer Grok แต่ไม่มี IDE presence) ยอมจ่าย 60x เพื่อ jumpstart ตรงนี้

ดีลนี้บีบ positioning ของ OpenAI และ Anthropic ทันที. OpenAI มี Codex แต่ใช้ผ่าน CLI/API เป็นหลัก ไม่ใช่ IDE-native. Anthropic มี Claude Code + Windsurf partnership แต่ไม่ได้เป็นเจ้าของ. **ถ้า Cursor กลายเป็น XAI/SpaceX-only backend ภายในปี** (ความเป็นไปได้สูงหลัง option exercise) = OpenAI/Anthropic เสีย distribution channel ใหญ่สุดใน AI coding tool — developer ที่ใช้ Cursor จะ default ไป Grok 5/XAI Max แทน GPT-5.5/Opus 4.7. คำถาม: Anthropic จะตอบด้วยการ acquire Windsurf (ที่ Google ถือ option อยู่) หรือ Zed? OpenAI จะเร่ง Codex IDE (มี rumor ที่ Dev Day)?

Microsoft โดนเจ็บหนักสุด. GitHub Copilot revenue $2B+ annualized แต่ growth flat ตั้งแต่ปลายปีที่แล้วเพราะ Cursor/Windsurf แย่งลูกค้า enterprise. ถ้าซื้อ Cursor ได้ = consolidate ตลาด + ลบคู่แข่ง. ไม่ได้ซื้อ = ต้อง build competing agent tier 1 ภายใน 6–12 เดือนไม่งั้น GitHub จะ repeat ประวัติ Internet Explorer — dominant แต่ค่อย ๆ โดนสิ่งใหม่กิน. Satya มี 60–90 วัน pivot strategy ก่อน Microsoft Build เปิดเดือนพ.ค.

Strategic read: **Musk ใช้ SpaceX balance sheet เล่นเกม M&A AI ที่ XAI เปล่งเสียงไม่ได้**. SpaceX IPO summer = war chest ใหม่ $50–100B — ถ้าเวิร์ก, คาดได้ว่าจะเห็น SpaceX ซื้อ AI company เพิ่มอีก 2–3 ราย ในช่วง 12 เดือน (เป้าหมาย likely: Perplexity ($25B tier), Harvey AI (legal), หรือ Abridge (healthcare))

## มุม OpenBridge

**ไม่ direct เกี่ยว แต่...** ดีลนี้ส่ง 3 signal ที่ OpenBridge ควร note. **หนึ่ง: AI coding/agent tooling = vertical ที่ valuation premium สูงสุด**. Cursor ที่ $1B revenue + $60B = 60x, Windsurf ที่ $400M revenue + Google $8B buyout = 20x, OpenBridge ที่ focus agent workflow orchestration ใช้ benchmark ไหน? ถ้า pitch VC รอบ Series A/B ในครึ่งปีหลัง ควร frame ว่าเป็น **"agent workflow tier"** ไม่ใช่ integration platform ทั่วไป — ราคาต่างกัน 3–5x. ทำงานคู่ storyline กับ Musk deal จะช่วยได้

**สอง: distribution + data moat = valuation driver มากกว่า model**. Cursor ไม่ได้ train model เอง — ใช้ GPT/Claude/Gemini backend, แต่ได้ valuation premium จาก **developer workflow data** + **distribution ใน IDE**. OpenBridge ที่ integrate LINE OA + FlowAccount + PEAK + Thai SaaS = กำลังสร้าง moat ชนิดเดียวกันสำหรับ Thai workflow — **data ของ Thai SMB operation flow ที่ไม่มีใครอื่นเก็บได้**. ต้องเริ่ม instrument logging + structured event capture ทุก workflow (consent-based, anonymized) ภายใน 30 วัน เพื่อสร้าง data flywheel ที่ Thai competitor ตามไม่ทัน

**สาม: Thai AI coding startup ควรเรียน lesson นี้**. ถ้าทีม Enabridge/OpenBridge รู้จัก Thai developer tooling startup (เช่น Codesphere Thailand, Trinsic, หรือ indie tool builders) ที่มี traction — timing นี้เหมาะที่สุดสำหรับ strategic partnership/acquisition talk. **Window 60–90 วัน** ก่อน SpaceX-Cursor deal จะ trigger secondary wave ที่ push valuation ขึ้นทั่ว
