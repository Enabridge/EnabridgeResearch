# Anthropic เปิด Claude Design — ขู่ Figma/Canva ตรง ๆ ด้วย Opus 4.7

## TL;DR
- Anthropic เปิด **Claude Design** เมื่อวาน (17 เม.ย.) — ขับด้วย Opus 4.7 vision, สร้าง prototype/slide/one-pager จาก prompt ตรง ๆ
- ใช้ในแผน Pro/Max/Team/Enterprise ไม่บวกเงิน, export เป็น Canva/PDF/PPTX/HTML ได้
- ตลาดตอบรับทันที — หุ้น Figma + Adobe ตกในวันเดียว

## เกิดอะไรขึ้น
17 เม.ย. 2026 Anthropic Labs เปิดตัว Claude Design ในโหมด research preview onboarding แรก Claude จะอ่าน codebase + design files ของทีมเพื่อสร้าง design system ของบริษัทขึ้นมาก่อน แล้วถึงเริ่มสร้างงาน

ผู้ใช้สามารถแก้แบบเป็น live editor — ปรับ spacing, สี, layout ผ่าน knob, comment inline ที่ element เฉพาะจุด, แล้วบอก Claude ให้ apply ทั้ง design ได้ Share เป็น internal URL ในองค์กร, save เป็น folder, หรือ export ออกเป็น Canva, PDF, PPTX, standalone HTML

ราคา: รวมอยู่ในแผน Claude Pro/Max/Team/Enterprise — ไม่ต้องจ่ายเพิ่ม ใช้ subscription limit ของแต่ละแผน Opus 4.7 API ราคาเท่าเดิม $5/M input, $25/M output

ตลาดตอบสนองทันที — Sherwood News รายงานว่าหุ้น Figma และ Adobe ตกในวันที่เปิดตัว

## ทำไมสำคัญ
นี่ไม่ใช่ "เครื่องมือสร้างรูป" แบบ Midjourney แต่เป็น vertical agent ที่ Anthropic ลงไปแย่ง surface ของ design tools ตรง ๆ — ตอกย้ำ pattern ที่เห็นมา 6 เดือน: foundation model lab จะลงมาทำ application layer เองในแนวที่ตัวเองได้เปรียบ (model + workflow integration)

ที่น่าสังเกตคือ "design system from codebase" — Claude เริ่มจากของจริงในองค์กร ไม่ใช่ template ทำให้ output ใช้ได้ทันทีในงาน production มากกว่า general AI design tool

## มุม OpenBridge
- ถ้า OpenBridge ทำ marketing/sales material — ลองให้ทีมใช้ Claude Design ทำ pitch deck/one-pager แทน Canva ดู เทียบ velocity จริง
- เป็นสัญญาณว่า "agent + workflow vertical" จะกินตลาด SaaS เดิมเร็วขึ้น — ถ้า OpenBridge เป็น integration platform คิดถึงว่า partner SaaS ของเราตัวไหนจะโดน disrupt ก่อน
